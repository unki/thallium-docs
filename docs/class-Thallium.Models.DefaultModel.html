Class DefaultModel

DefaultModel is an abstract class that is used by all the other Thallium Models.

It declares some common methods, properties and constants.

Direct known subclasses

Skeleton\Models\DefaultModel, Thallium\Models\AuditEntryModel, Thallium\Models\AuditLogModel, Thallium\Models\JobModel, Thallium\Models\JobsModel, Thallium\Models\MessageBusModel, Thallium\Models\MessageModel
